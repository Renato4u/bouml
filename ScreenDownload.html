Descargas e Instalación por Sistema

Microsoft Windows

BOUML es compatible con Windows desde la versión 7 hasta la 11 (32 y 64 bits). El instalador incluye automáticamente las dependencias de Qt 4.8 y los plug-outs básicos.

Instalador ejecutable:
Bouml_7.11.patch4_setup.exe
Si encuentras problemas al guardar proyectos, haz clic derecho y selecciona "Ejecutar como administrador".

Sistemas basados en Debian (Ubuntu, Mint, Raspbian)

Para arquitecturas amd64, i386, armhf y arm64. Debido a que el servidor utiliza SSL, se requiere el transporte HTTPS para APT.

# 1. Instalar soporte HTTPS para el gestor de paquetes sudo apt-get install apt-transport-https # 2. Añadir la llave de firma oficial de BOUML wget -q https://www.bouml.fr/bouml_key.asc -O- | sudo apt-key add - # 3. Actualizar e instalar el paquete sudo apt-get update sudo apt-get install bouml

Distribuciones RPM (CentOS, Fedora, Mageia, openSUSE)

Para sistemas RedHat-like. Si utilizas una versión reciente de Fedora o CentOS Stream, asegúrate de habilitar los repositorios de compatibilidad para Qt 4.8.

CentOS 7 / Stream 8 & 9:
Usa el paquete bouml-7.11.patch4-1.x86_64.rpm. Instalación vía: yum localinstall bouml...rpm
Fedora & Rosa Linux
Los paquetes de CentOS son 100% compatibles. En Rosa, se recomienda usar el instalador urpmi.
openSUSE Leap / Tumbleweed:
Instalación mediante zypper install tras descargar el RPM correspondiente a x86_64.

Otros Sistemas Unix (Slackware, Gentoo, BSD)

Información detallada sobre portabilidad y binarios para sistemas no basados en Debian o RPM.

Slackware Linux:
Para Slackware 14 y superiores, usar los binarios compilados para CentOS 6.
Gentoo / OpenBSD / FreeBSD:
Paquetes binarios disponibles y soporte a partir de OpenBSD 6.x.
Archlinux / Manjaro:
Usa el repositorio AUR. El comando estándar es yay -S bouml.

Apple MacOS X

Compatible con procesadores Intel. El paquete viene pre-compilado en formato de imagen de disco.

Imagen DMG: bouml_patch4_macosx.dmg
Arrastra el icono de BOUML a tu carpeta de Aplicaciones para completar la instalación.
