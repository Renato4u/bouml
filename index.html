BOUML

Overview

BOUML es una herramienta UML 2 gratuita que incluye un modelador que te permite especificar y generar codigo en C++, Java, Idl, Php, Python y MySQL.

  • Desde la version 7.0 BOUML vuelve a ser un software libre.
  • BOUML se ejecuta en Windows, Linux y MacOS X.
  • BOUML es muy rápido y requiere poca memoria para gestionar miles de clases, ver benchmark.
  • BOUML es extensible, y las herramientas externas llamadas plug-outs pueden escribirse en C++ o Java utilizando BOUML para su definición como cualquier otro programa. Los generadores de código, inversiones y rondas son algunos de los plug-outs predefinidos incluidos en la distribucion de BOUML. Ver
